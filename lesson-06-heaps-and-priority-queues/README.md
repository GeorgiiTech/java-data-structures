# Lesson 6: Heaps and Priority Queues - Min-Heap, Max-Heap, and Heap Sort

> Back to Course: https://georgii.tech/courses/java-data-structures-a-deep-dive-with-visualizations

## Overview

Heaps are complete binary trees stored efficiently in arrays. The Min-Heap guarantees the smallest element is always at the top, while the Max-Heap keeps the largest. Priority queues are their direct application - used in Dijkstra, task scheduling, and more.

## What you'll learn

- Heap property and complete binary tree structure
- Insert and extract operations with sift-up/sift-down
- Min-Heap and Max-Heap implementation
- Java PriorityQueue usage
- Heap Sort algorithm in O(n log n)
