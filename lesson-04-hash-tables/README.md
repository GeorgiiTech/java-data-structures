# Lesson 4: Hash Tables — How Hashing Works and HashMap Internals

> 📚 [Back to Course](https://georgii.tech/courses/java-data-structures-a-deep-dive-with-visualizations) | [Course Repo](../../README.md)

## Overview

Unlock the mystery behind O(1) lookups. In this lesson you learn how hash functions convert keys into array indices, how Java's HashMap handles collisions with chaining, and why load factor matters for performance.

## What you'll learn

- How hash functions compute array indices
- Collision resolution: separate chaining vs open addressing
- Java HashMap internal structure and resize behaviour
- Time complexity of put, get, and remove operations
- When to use HashMap, LinkedHashMap, or TreeMap

## Run the code

```bash
javac HashTables.java
java HashTables
```
