# Lesson 2: Linked Lists — Singly, Doubly, and Circular

> 📚 [Back to Course](https://georgii.tech/courses/java-data-structures-a-deep-dive-with-visualizations) | [Course Repo](../../README.md)

See exactly how linked list nodes connect in memory, how every operation works step-by-step with pointer diagrams, and understand the real trade-offs vs arrays.

## Big-O Summary
| Operation      | Singly Linked | Array |
|----------------|---------------|-------|
| Access by index | O(n)         | O(1)  |
| Insert at head  | O(1)         | O(n)  |
| Insert at tail  | O(1)*        | O(1)  |
| Delete at head  | O(1)         | O(n)  |
| Search          | O(n)         | O(n)  |

## Run the code
